Permisos de archivos en Linux

Descripción
Este proyecto consiste en analizar y corregir permisos de archivos en Linux para mejorar la seguridad del sistema.

---

Objetivos
- Revisar permisos con `ls -la`
- Detectar accesos inseguros
- Corregir permisos con `chmod`

---

Proceso
Se inspeccionaron archivos del sistema y se corrigieron permisos incorrectos para eliminar accesos no autorizados.

---

 Resultado
Se logró una configuración segura de permisos aplicando el principio de mínimo privilegio.

---

Aprendizaje
Control de permisos en Linux y seguridad básica de archivos.
